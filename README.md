# Android With Java Course Project

This repository contains the project I completed as part of my Android development course using Java. It serves as a practical demonstration of various Android components, APIs, and development concepts.

## About This Project

This application is a collection of several mini-apps, each designed to explore a specific feature of the Android framework. It showcases my ability to build a multi-activity application and integrate different functionalities into a single, cohesive user experience.

## Technologies and Concepts Learned

*   **Core Components:** `Activity`, `Intent`, `View`, `Button`
*   **User Interface:** XML Layouts (`ConstraintLayout`), `ViewCompat`, `EdgeToEdge` UI
*   **Event Handling:** `OnClickListener` and event-driven programming.
*   **Third-Party Libraries:**
    *   Google Play Services for Maps
    *   Retrofit for networking
    *   ExoPlayer for media
*   **Version Control:** All project history is managed with Git and hosted on GitHub.

## How To Run

1.  Clone the repository: `git clone https://github.com/YourUsername/AndroidWithJavaCourse.git`
2.  Open the project in Android Studio.
